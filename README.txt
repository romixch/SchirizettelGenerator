Diese Applikation hilft dir beim Erstellen von Korbball-Resultate-Zettel.
Als Voraussetzung muss Java ab der Version 7 installiert sein. Um die
Resultate-Zettel zu erstellen, öffne einfach die Datei schirizettel.jar
mit Java. Ein Fenster öffnet sich. Darin kannst du folgende drei Dinge
auswählen:

* Report:  Vorlage, wie die Zettel aussehen sollen. Es sind bereits zwei
           Vorlagen mit dabei.
* Daten:   Eine CSV-Datei, welche die zu druckenden Daten enthällt.
* Ausgabe: in diese PDF-Datei wird das Resultat geschrieben.

Probleme:
Wenn das produzierte PDF leer ist, solltest du das Format deiner csv-
Datei überprüfen.

Tipp:
Mit Eclipse-Birt (http://www.eclipse.org/birt/) kannst du die Vorlage
anpassen.
